## Notas para el Design System
- Todos los SVGs están construidos con las coordenadas exactas del manual de marca
- Para Figma: importar los SVG directamente como componentes
- Para Canva: subir los SVG como elementos de marca en el Brand Kit
- Los logos en SVG pueden exportarse a PNG desde cualquier navegador (clic derecho → guardar imagen)
